How to redeem your discounted coupon codes:

  1. We suggest that you take a screenshot of the products and coupons listed above. It will make things a lot easier for you and the cashier.
  2. Be sure to pick exactly the same item from the shelves to what is defined on the list (check that the weight etc is not different).
  3. Present your list of coupon codes to the cashier when you pay.
  4. The cashier must enter the coupon codes like a product barcode that does not scan - each coupon code can be used to claim the discount on any products with the same value eg the cashier will only need to enter ONE code for all R3 coupons and ONE code for all R20 coupons when you check out. They will not need to enter multiple codes for the same discount value.
  5. The cashier must press “enter” and then “total” after each coupon code entered.
  6. If you have any difficulty redeeming your discount codes in store, please sms the word “Gymco” to 31239 for an agent to call you back - this is a FREE sms service.

Terms and conditions apply.

  1. Your coupon codes are valid for 30 days from coupon selection as reflected in the coupon check out section.
  2. Your coupon codes will remain in your basket for as long as they are valid.
  3. All discounts are guaranteed for all products you selected and present in store, subject to stock availability.
